# FZburner-CC
给大鱼CC 3D打印机设计的一款打印头
> Different designs of the burner for the DaYu CC 3D parinter

### 主要功能
> - [x] 近程挤出
> - [x] E3D V6热端
> - [x] 4010热端风扇
> - [x] 4020物料风扇
> - [x] BLTouch安装位
> - [x] klicky probe安装位
> - [x] ADXL 345安装位
